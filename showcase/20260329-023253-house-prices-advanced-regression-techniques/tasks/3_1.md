在本项目中，我们对三种基础线性回归模型（Lasso、Ridge 和 ElasticNet）进行了训练与超参数调优。为了处理潜在的离群点，我们在模型管道中集成了 `RobustScaler`。由于目标变量 `SalePrice` 在预处理阶段已进行 `log1p` 变换，交叉验证计算得到的 RMSE 实际上等同于原始房价的 RMSLE。

### 1. 模型训练与调优结果

通过 5 折交叉验证（GridSearchCV）获得的最佳超参数及对应得分如下：

| 模型 | 最佳超参数 | 验证集 RMSE (RMSLE) |
| :--- | :--- | :--- |
| **Lasso** | `alpha: 0.0005` | **0.111995** |
| **Ridge** | `alpha: 15` | 0.113156 |
| **ElasticNet** | `alpha: 0.0005`, `l1_ratio: 0.9` | 0.112031 |

**结论**：在三种线性模型中，**Lasso** 表现最佳，其 RMSLE 最低。ElasticNet 在 `l1_ratio=0.9` 时表现与 Lasso 极度接近，这表明模型倾向于更强的 L1 正则化来筛选特征。

### 2. 生成的产出文件

*   **`linear_models_results.json`**: 包含三种模型的最佳超参数及 CV 均方根误差。
*   **`linear_models_oof.csv`**: 保存了训练集的 Out-of-fold (OOF) 预测值，包含 `Id`, `Ridge`, `Lasso`, `ElasticNet` 四列，可用于后续的集成学习（Stacking）。
*   **`best_score.json`**: 记录了目前为止的最佳模型及其得分：
    ```json
    {
        "metric": "RMSLE",
        "score": 0.11199515156671268,
        "model": "Lasso",
        "details": "5-fold CV, tuned Lasso"
    }
    ```

### 3. 后续工作建议
鉴于线性模型已达到约 0.112 的 RMSLE，接下来可以引入更复杂的非线性模型（如 Gradient Boosting 家族），并结合这些线性模型的 OOF 预测进行集成，以进一步提升预测精度。
