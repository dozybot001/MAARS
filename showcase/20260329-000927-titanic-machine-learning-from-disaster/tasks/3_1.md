基于在任务 2 中生成的预处理数据，我使用 **随机森林 (Random Forest)** 分类器对泰坦尼克号数据集进行了训练，并通过 **5 折交叉验证 (5-fold Cross-Validation)** 评估了模型的性能。

### 1. 实验设置

*   **模型**: `RandomForestClassifier` (使用默认参数: `n_estimators=100`)
*   **特征**: 包含经 One-Hot 编码后的称呼 (Title)、登船港口 (Embarked)，以及标准化处理后的年龄 (Age)、票价 (Fare)、客舱等级 (Pclass)、家庭规模 (FamilySize) 和是否有客舱号 (HasCabin)。
*   **验证方法**: 5 折交叉验证，以确保结果的稳健性并减少对单一训练集/测试集划分的偏差。
*   **随机状态**: 设为 `42` 以保证实验的可重复性。

### 2. 评估结果

交叉验证的准确率如下：

| 折数 (Fold) | 准确率 (Accuracy) |
| :--- | :--- |
| Fold 1 | 82.12% |
| Fold 2 | 77.53% |
| Fold 3 | 84.27% |
| Fold 4 | 76.97% |
| Fold 5 | 83.71% |

**综合指标统计：**

*   **平均准确率 (Mean Accuracy):** **80.92%**
*   **标准差 (Standard Deviation):** **3.08%**

### 3. 结论与分析

随机森林模型在预处理后的数据上表现良好，平均准确率达到了约 **80.92%**。各折之间的准确率标准差为 3.08%，显示出模型对数据的波动具有一定的敏感性，但整体性能表现稳健。
