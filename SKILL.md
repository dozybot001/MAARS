# Project Skills

项目内的流程型说明和可复用 skill 统一收纳在根目录 [`skills/`](./skills) 下。

当前条目：

- `skills/release/`: MAARS 仓库发版流程与双语 release notes 模板

建议约定：

- 一个主题一个目录
- 每个主题目录的主入口命名为 `SKILL.md`
- 可选补充 `agents/`、`references/`、`examples/`
- 内容尽量写成仓库定制版，不复制通用平台文档

后续像 `MAARS CLI`、部署流程、运维手册，也可以按同样结构继续补到 `skills/` 下。
