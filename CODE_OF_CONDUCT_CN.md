# 行为准则

中文 | [English](CODE_OF_CONDUCT.md)

### 我们的承诺

我们致力于让每个人都能在友好的环境中参与本项目，不论其背景或身份。

### 行为标准

**积极行为：**

- 使用友善和包容的语言
- 尊重不同的观点和经验
- 虚心接受建设性的批评
- 关注对社区最有利的事情

**不可接受的行为：**

- 骚扰、攻击性言论或贬损评论
- 未经同意发布他人的私人信息
- 其他被合理认为不适当的行为

### 执行

如遇到不当行为，可通过提交 issue 或联系维护者进行举报。所有投诉都会被公正审查和处理。

### 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 2.1 版本。
